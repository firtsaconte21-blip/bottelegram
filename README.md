# Mile Finder Bot

## Project info

A React application built with Vite, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
4. Build for production:
```bash
npm run build
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
